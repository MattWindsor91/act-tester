# NAME

act-tester-query - performs human-readable queries on a plan file

# SYNOPSIS

act-tester-query

```
[-i]=[value]
```

**Usage**:

```
act-tester-query [GLOBAL OPTIONS] command [COMMAND OPTIONS] [ARGUMENTS...]
```

# GLOBAL OPTIONS

**-i**="": read from this plan `file` instead of stdin
