% c4t-mkdoc 8

# NAME

c4t-mkdoc - makes documentation for c4t commands

# SYNOPSIS

c4t-mkdoc

```
[--help|-h]
[-d]=[value]
```

**Usage**:

```
c4t-mkdoc [GLOBAL OPTIONS] command [COMMAND OPTIONS] [ARGUMENTS...]
```

# GLOBAL OPTIONS

**--help, -h**: show help

**-d**="": `directory` to which outputs will be written (default: docs)
