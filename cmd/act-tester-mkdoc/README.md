# NAME

act-tester-mkdoc - makes documentation for act-tester commands

# SYNOPSIS

act-tester-mkdoc

```
[--help|-h]
[-d]=[value]
```

**Usage**:

```
act-tester-mkdoc [GLOBAL OPTIONS] command [COMMAND OPTIONS] [ARGUMENTS...]
```

# GLOBAL OPTIONS

**--help, -h**: show help

**-d**="": `directory` to which outputs will be written (default: docs)
